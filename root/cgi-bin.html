CGI Input Form

How it works:

  • POST method: Data is sent in the request body and read from stdin by the CGI script
  • GET method: Data is sent as a query string (?input=your+text) and read from QUERY_STRING environment variable
  • The CGI script (/cgi-bin/echo.py) processes the input and generates an HTML response

Try it: Enter some text and click either button to see how the CGI script handles your input!
